Procedural Tree Growth Animation (Blender)

This project is a procedural Blender animation created entirely using Python. It visualises seeds growing into trees over time, combined with rain, lighting, and a hidden Easter egg within the scene.

Project Overview

The animation begins with a flat ground plane. Seeds appear at staggered intervals, grow into tall tree trunks, and finally form leafy canopies. Rain falls continuously while the sun gradually brightens the environment.

All objects, materials, animations, and keyframes are generated automatically using the Blender bpy API.

Key Features

  • Procedural creation of multiple trees
  • Animated tree growth using keyframes
  • Staggered timings for natural effect
  • Rain simulation with random motion
  • Sunlight animation
  • Hidden Easter egg embedded in scene

Scene Elements

Ground

A large green plane forms the base of the scene.

Trees

Each tree consists of a seed, a trunk and a leaf canopy grown in sequence.

Rain

Rain drops fall from random heights and disappear upon reaching ground level.

Lighting and Camera

A sun light source gradually increases in intensity while the camera is positioned for a wide view of the scene.

Easter Egg

A bright red “67” is hidden in the scene using primitive shapes.

How to Run

  1. Open Blender
  2. Switch to the Scripting workspace
  3. Paste the Python script into the text editor
  4. Press Run Script
  5. Play the timeline to view the animation
