Dual Sensor Leak Detector (Arduino)

This project demonstrates a simple leak detection system using two analogue sensors connected to an Arduino. If either sensor detects a value above a defined threshold, an LED indicator is activated.

Project Overview

The system continuously monitors two analogue input pins. When a potential leak is detected by either sensor, the LED turns on and remains illuminated for a fixed duration, even if the sensor value drops immediately.

Sensor readings are also sent to the Serial Monitor for debugging and observation.

Key Features

  • Two analogue leak sensors
  • Adjustable sensitivity threshold
  • LED alert with timed hold
  • Serial output for monitoring
  • Simple and reliable logic

How It Works

  1. Both sensors are read continuously using analogue inputs.
  2. If either sensor exceeds the threshold, the LED turns on.
  3. The LED remains on for three seconds.
  4. After the timer expires, the LED turns off automatically.

Applications

  • Water leak detection
  • Moisture monitoring
  • Safety and alert systems
  • Educational Arduino projects
